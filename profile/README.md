# RunminG-Lab
RunminG-Lab 是由来自南京大学、东南大学、华东师范大学、南京师范大学、中山大学等实验室和高校的研究人员，共同维护的开源代码贡献组织, 我们主要关注 **大模型、软件工程**等领域的实用工具和研究原型。



## 我们关注什么

- AI 工具与 AI Agent
- 软件工程研究原型
- Vibe Coding 项目与开发者工具
- 论文阅读、审稿与科研工作流工具
- 课程项目、学习资源与可复用代码
